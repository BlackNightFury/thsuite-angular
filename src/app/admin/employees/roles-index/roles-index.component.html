Role Description
Owner/Administrator This is a master role with full access to all administrative and point of sale settings. This role cannot be edited. EDIT
Manager This role is intended for managers that supervise a store and have partial administrative access to oversee budtenders and the performance of their store. This is a default role that cannot be removed. EDIT
Budtender This role is intended for budtenders that conduct sales transactions and have very limited administrative access. This is a default role that cannot be removed. EDIT
